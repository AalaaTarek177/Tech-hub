Game Development Track

🎮 Turn your ideas into playable games! .

What You Will Learn

  • 🎮 Understand how games are built from scratch.
  • 🎨 Learn basic game design and mechanics.
  • 🧱 Create 2D games using beginner-friendly tools.
  • 💻 Practice coding logic in C# or JavaScript.
  • 🔊 Add sound, effects, and interactions to your game.
  • 🚀 Publish your first playable game online!.

How You Will Learn

  • 📚 Short video lessons with simple explanations.
  • 🧩 Interactive projects after each module.
  • 🕹 Build mini-games like Flappy Bird, Memory Match.
  • 🧪 Hands-on practice using game engines.
  • 👨‍🏫 Mentorship and guidance for every step.
  • ✅ Quizzes and challenges to test progress.

Skills You Will Acquire

Before starting the Game-development track, we recommend having the following:

  • ✅ Basic computer usage (mouse, keyboard, folders).
  • ✅ Logical thinking & creativity.
  • ✅ No prior coding experience needed (but helpful!).
  • ✅ Recommended: 4GB+ RAM, Windows/Mac PC.
  • ✅ Internet access to download tools or use web engines.
